### A Todo-App using react, express, nodejs, mongodb, tailwindcss, shadcn 

# Features: 
- Add a Todo
- Edit Todo
- Delete Todo
- Light-Mode / Dark-Mode

## Run 

`cd front-end`

`npm i`

`npm start`

`cd backend`

create an .env file and paste your MongoDB cluster url as #MONGODB_URL = url

`npm i`

`npm start`
